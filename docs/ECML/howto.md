# How to install the mod
## Follow these steps:
<b>1.</b> Get <a href="https://github.com/LavaGang/MelonLoader/releases/download/v0.5.7/MelonLoader.x86.zip">MelonLoader v0.5.7 (x86)</a>.<br>
<b>2.</b> After unzipping the downloaded file directly into the game's folder, launch the game.<br>
<b>3.</b> Now close the game and <a href="https://github.com/Code4Creations/ECML/releases">download the mod</a>.<br>
<b>4.</b> Unzip the mod's contents into the Mods folder inside the game's folder, if the Mods folder isn't there, create it.<br>
<b>5.</b> You did it! Run the game and have fun!
